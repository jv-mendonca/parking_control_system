# 🚘 Parking Control System (Java)

Sistema de controle de estacionamento desenvolvido em Java, com foco em **boas práticas de Programação Orientada a Objetos (POO)** e manipulação de dados.

---

## 🔍 Sobre o projeto

Este projeto simula o funcionamento de um estacionamento, permitindo registrar a entrada e saída de veículos, controlar tickets e armazenar informações em arquivo.

Foi criado como prática para consolidar conceitos fundamentais de desenvolvimento backend.

---

## ⚙️ Funcionalidades

✔ Registro de entrada de veículos
✔ Registro de saída com horário automático
✔ Identificação por ticket
✔ Armazenamento de dados em `.csv`
✔ Busca de registros por ticket

---

## 🧱 Arquitetura

O projeto está organizado em camadas:

* `application` → execução do sistema
* `entities` → regras de negócio
* `src` → estrutura principal do código
